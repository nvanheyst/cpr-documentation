#### Package and Setup
The MicroStrain IMU devices use the `microstain_inertial` ROS 2 package. The driver is open source, maintained by MicroStrain, and hosted on [GitHub](https://github.com/LORD-MicroStrain/microstrain_inertial).  The `device_type` will default to the `GX5` but `GX3`, `GX4`, `GX5`, `CX5`, `RQ1`, `GQ7` and `GV7` are supported.

For more specifics on the way Clearpath's configuration system launches the `microstrain_inertial` driver, see the MicroStrain GX [launch file](https://github.com/clearpathrobotics/clearpath_robot/blob/jazzy/clearpath_sensors/launch/microstrain_imu.launch.py) and the [default parameter file](https://github.com/clearpathrobotics/clearpath_robot/blob/jazzy/clearpath_sensors/config/microstrain_imu.yaml) in `clearpath_sensors`.

#### Device Path
The `port` ROS parameter must be set to the device path of the MicroStrain device. The `microstrain_inertial` package installs its UDEV rule, which will add the `/dev/microstrain_main` device.
