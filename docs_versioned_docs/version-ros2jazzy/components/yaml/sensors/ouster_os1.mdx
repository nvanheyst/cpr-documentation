#### Package and Setup
Ouster lidars use the [`ouster_ros`](https://github.com/ouster-lidar/ouster-ros) driver. This is
an open-source driver, maintained by the manufacturer.

For more specifics on the way Clearpath's configuration system launches the `ouster_ros` nodes,
see the Ouster [launch file](https://github.com/clearpathrobotics/clearpath_robot/blob/jazzy/clearpath_sensors/launch/ouster_os1.launch.py)
and the [default parameter file](https://github.com/clearpathrobotics/clearpath_robot/blob/jazzy/clearpath_sensors/config/ouster_os1.yaml)
in `clearpath_sensors`.

#### Base & Cap
The Ouster lidar can be configured with different base and cap options:

* `base_type`: one of `base` (default) or `none`.
* `cap_type`: one of `halo` (default) or `fins`.
