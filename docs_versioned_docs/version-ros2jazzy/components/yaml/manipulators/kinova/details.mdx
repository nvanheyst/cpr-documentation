### Package and Setup {#kinova_package_and_setup}
Kinova manipulators uses the `kortex_description` and `kortex_driver` ROS 2 Packages. The driver is open source, maintained by Kinova Robotics, and hosted on [GitHub](https://github.com/Kinovarobotics/ros2_kortex).

For more specifics on the way Clearpath's configuration system adds the manipulator to the robot description, see the Kinova's [description files](https://github.com/clearpathrobotics/clearpath_common/blob/jazzy/clearpath_manipulators_description/urdf/arm/) in `clearpath_manipulators_description`.

### Device IP and Port {#kinova_param_ip_port}
The `ip` and `port` parameters must be set to the corresponding values set on the manipulators configuration page.
