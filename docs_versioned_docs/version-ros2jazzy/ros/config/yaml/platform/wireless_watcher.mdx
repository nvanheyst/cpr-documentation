By default, all Clearpath platforms launch a [Wireless Watcher node](https://github.com/clearpathrobotics/wireless).
This node monitors the computer's integrated wireless interface connection. Some applications may not use this integrated wireless connection,
in which case the wireless watcher node can be disabled although it can be left enabled without negative ramifications.

|           Key           | Value / Datatype | Description                                                    |
| :---------------------: | :--------------: | -------------------------------------------------------------- |
| enable_wireless_watcher |     boolean      | Enables or disables the wireless watcher node. Default: `true` |

For example, to disable the wireless watcher node:

```yaml
platform:
  enable_wireless_watcher: false
```
