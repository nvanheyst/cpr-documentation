Clearpath robots using ROS 2 Jazzy use [Ubuntu 24.04 LTS "Noble Numbat"](https://releases.ubuntu.com/noble/).
While other operating systems are supported by ROS, we highly recommend using Ubuntu.

## Upgrading

If your robot has an older version of ROS (e.g. ROS 2 Humble or ROS 1 Noetic) it can be [upgraded](upgrading) to
ROS 2 Jazzy.

## Robot installation

All Clearpath robots ship with the operating system and core components installed. Should you wish to
[reinstall these components](robot) you may.

## Offboard computer

An [offboard computer](offboard_pc) can be used to visualise and interact with the robot. The robot can also be simulated
on an offboard computer to safely test software before deploying to a physical robot.

## Joystick controller pairing

When upgrading or reinstalling the operating system on your robot, you will need to [re-pair](controller)
the game controller that shipped with your robot.
