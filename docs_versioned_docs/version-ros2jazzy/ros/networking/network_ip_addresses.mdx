:::note

All Clearpath robots use the `192.168.131.0/24` subnet for internal networking. This includes
communication between the computer(s), MCU, sensors, manipulators, and other network-based components.

:::

The following table defines the standard IP addresses Clearpath uses for custom robot configurations.
The details of your specific integration should have been included with your robot as a _robosmiths memo_
stating the device, serial number, and IP address.

If you are integrating your own sensors, manipulators, or additional computers, we recommend
following the table below. Make sure you do not use the same IP address for multiple components,
as this will cause errors.
