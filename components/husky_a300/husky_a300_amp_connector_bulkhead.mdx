To make a Husky AMP, we take a Husky A300, remove the Top Plate, and then install an enclosure on top of the base robot.
We added a connector bulkhead between the base robot and the enclosure to keep cable routings tidy, and make the maintenance process quick.
The connectors on this bulkhead are extension cables from existing [Husky A300 interfaces](/docs_robots/outdoor_robots/husky/a300/integration_husky/#system-interface-circuit-board-connector-summary), which are listed below.

| Row | Interface                               | Purpose for AMP                                                                                                                                                                                               | Connector on the bulkhead | Mating connector  |
| :-: | :-------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | :------------------------ | :---------------- |
|  1  | AUX Outputs                             | Controlling Lights and Sounder                                                                                                                                                                                | Molex 0039012106          | Molex 0039012105  |
|  2  | Network Switch Power                    | Powering the AMP's network switch, and access point                                                                                                                                                           | Molex 0039012026          | Molex 0039012025  |
|  3  | Emergency Stop Buttons                  | 2 additional Emergency Stop Buttons are included with AMP, compared to the base Husky A300.                                                                                                                   | TE 794615-4               | TE 794617-4       |
|  4  | User Power Connections                  | The bulkhead includes 8 cutouts for the 8 User Power Connectors from the base Husky A300 System Interface Assembly. These may be populated with an extension cable, if required for your AMP's configuration. | Molex 0039012086          | Molex 0039012080  |
|  5  | High Power Expansion Kit _(Large VBAT)_ | This is not required for a standard Husky AMP. We provide this interface if the High Power Expansion Kit is included in the base Husky A300.                                                                  | Anderson 992G5-BK         | Anderson 992G5-BK |
|  6  | RJ45 Ethernet                           | This connects the base Husky A300's network switch to the AMP enclosure's network switch.                                                                                                                     | RJ45 Port                 | RJ45 Jack         |

:::info

We add Bluetooth and Wireless E-stop antenna cables on top of the bulkhead using cable ties.
This keeps the antennas electrically isolated from the robot's chassis.

:::tip

You can add custom cables on top of the bulkhead using tie wraps.
You should include an inline connector at this interface for your custom cables, so the enclosure can easily be removed from the base Husky A300 for maintenance.

:::
